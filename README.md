# Steps To Run
 1. Clone the git repo
    ```sh 
    https://github.com/ardentfiery/vrit.git
    ```
 2. Change directory to the project folder
    ```sh
    cd vrit
    ```
 3. Install dependencies
    ```sh
    npm install
    ```
 5. Run Project
    ```sh
    npm run dev
    ```
